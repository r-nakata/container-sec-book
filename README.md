# コンテナセキュリティ サンプルコード集

本書で使用するサンプルコードを章ごとにまとめています。
各ファイルは「章番号.通し番号_タイトル.txt」の形式で命名されています。

## ファイル構成

```
book-samples/
├── README.md              # このファイル
├── CODE_LIST.md           # コード一覧
├── chap02/               # 第2章 Dockerの特徴と基本操作（20ファイル）
├── chap03/               # 第3章 コンテナの運用と管理（23ファイル）
├── chap04/               # 第4章 コンテナエコシステムの広がり（10ファイル）
├── chap05/               # 第5章 コンテナセキュリティ（54ファイル）
├── chap06/               # 第6章 Kubernetesの基礎（13ファイル）
├── chap07/               # 第7章 マイクロサービスとコンテナ（3ファイル）
└── chap08/               # 第8章 オーケストレーションとマイクロサービスのセキュリティ（27ファイル）
```

総計：150ファイル

## 動作環境

- Docker 24.0以上
- Docker Compose 2.0以上
- Kubernetes 1.28以上（第6章、第8章）
