# 5.24 起動しているコンテナへの帯域制限設定
# 『コンテナセキュリティ』サンプルコード

# 基本的な送信帯域制限（1Mbps）
docker exec myapp tc qdisc add dev eth0 root tbf rate 1mbit burst 32kbit latency 400ms

# より厳しい制限（512Kbps）
docker exec myapp tc qdisc add dev eth0 root tbf rate 512kbit burst 16kbit latency 200ms

# 高帯域環境での制限（100Mbps）
docker exec myapp tc qdisc add dev eth0 root tbf rate 100mbit burst 1mbit latency 50ms
